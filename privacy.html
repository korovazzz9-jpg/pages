Privacy Policy
Last updated: June 25, 2026

1. Overview

60SecFacts ("App") is an automated video publishing tool. This Privacy Policy explains how we handle information in connection with the App.

2. Information We Collect

The App does not collect personal information from end users. The App uses TikTok's Content Posting API solely to publish video content to TikTok accounts authorized by the account owner.

3. TikTok API Data

The App accesses TikTok account data (access tokens) only as necessary to post videos on behalf of the authorized account owner. We do not store, share, or sell any TikTok user data.

4. Data Storage

Access tokens are stored securely as encrypted environment variables in GitHub Actions and are never exposed publicly.

5. Third-Party Services

The App uses the following third-party services:

  • TikTok Content Posting API
  • Anthropic Claude API (AI content generation)
  • Pexels / Pixabay (stock video footage)
  • Cloudinary (temporary video hosting)

Each service has its own privacy policy governing their data practices.

6. Data Retention

We do not retain any user data. Temporary video files used during processing are deleted immediately after upload.
